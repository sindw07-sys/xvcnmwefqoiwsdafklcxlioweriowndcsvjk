# UpLog

UpLog는 **웹앱 우선(Web-first)** 으로 시작하는 캘린더 중심 프로젝트입니다.

## 프로젝트 핵심 방향
- 앱 표시 이름: **UpLog**
- 프로젝트 폴더명: **uplog**
- 1차 목표: **월간 캘린더 기반의 사용 흐름** 완성
- 개발 우선순위: **웹앱 우선**
- 범위 제한: notes / todo / AI 기능은 초기 버전에서 보류

## 현재 단계
이 저장소는 UpLog 초기 인수인계 문서 중심으로 세팅되었으며, 코드 작성 전 아래 문서들을 기준으로 프로젝트 방향을 확정합니다.
- `PROJECT_GUIDE.md`
- `DESIGN_RULES.md`
- `BIGDATA_UI_RULES.md`
- `TODO.md`

## 초기 제품 비전
- 사용자가 월간 뷰에서 날짜 단위로 일정을 빠르게 확인/기록
- 복잡한 기능보다 **신뢰성 있는 달력 경험**을 먼저 제공
- 이후 점진적으로 기능 확장 (notes/todo/AI는 후속 단계)

## 실행 방법
```bash
npm install
npm run dev
```

GitHub Actions에서 build 확인 가능.
